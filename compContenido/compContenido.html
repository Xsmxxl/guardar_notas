Webapp para guardar notas.

Ésta aplicación web ha sido desarrollada cómo proyecto para crear un framework que utilice un sólo index conformado por varios componentes que pueden ser el título, el login el contenido incluso el footer cómo se ha hecho con éste ejemplo. También se está implementando un método que permita compartir variables entre componentes en tiempo real sin necesidad de recargar la página y haciendo que ésta cargue un poco más rapido por que no estára toda estilizada sino que a medida que se interactue se iran acoplando los demás componentes ya preestablecidos.

Template desarrollada con Bootstrap, por Xsmxxl.
